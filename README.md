## Project Overview
An end-to-end manufacturing dashboard built to track 30+ orders across 14 customers and 3 distinct industries. This project focuses on data cleaning, KPI visualization, and interactive reporting.

<img width="991" height="668" alt="dashboard_preview" src="https://github.com/user-attachments/assets/c10005a9-dc01-48ef-b5fb-190e907436ab" />

## Key Features
* **Interactive Filters:** Date range and customer-specific drill-downs.
* **Automated Logic:** Built-in SUMIF, COUNTIF, and VLOOKUP formulas for real-time data processing.
* **Data Cleaning:** Handled missing values, duplicates, and formatting issues from raw manufacturing logs.
* **Visualizations:** 8+ charts including KPIs, Pie Charts, Bar Charts, and Line Charts.

## Tech Stack
* **Visualization:** Looker Studio
* **Database/Logic:** Google Sheets & Excel
* **App Integration:** AppSheet
